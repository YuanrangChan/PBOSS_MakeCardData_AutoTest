# PBOSS_MakeCardData_AutoTest

项目背景：
   项目上的卡资源申请需要发报文生成订单，多次加密上传文件并报文反馈，流程相当繁琐耗时，本项目旨在实现一键完成所有流程，包括报文生成发送，文件生成加密上传反馈报文，以及最终下发的卡资源及索引文件的下载。
